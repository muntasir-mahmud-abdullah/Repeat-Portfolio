Project Name: Career Counseling Website

Main Technology Stack Used

React.js, Firebase Authentication, React Router, Tailwind CSS, DaisyUI, Swiper.js

Brief Description

This project is a Career Counseling Website designed to help users make informed career decisions through personalized guidance, job matching, resume reviews, and interview preparation.
